= ABOUT NOVA =

* Contributors: play0ad.com/humans.txt

* Description:
Nova was a true collaboration between artists and developers. Ever
since the initial mockup was presented, over time, it  completely
evolved in order to adapt to the requests of the WFG team. From 
the beginning, Nova was meant to be modular and easily modified.

The theme was based on the WordPress 2011 theme. The 2011 theme
is packed with many features and Nova has been minified to contain
only what is necessary and just a bit more. The stylesheet has been
compressed and comments have been disabled. Many different post
types have also been removed. Custom pages have been added as well.
